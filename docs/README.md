# 和牛焼肉 炎座 - 架空の焼肉店サイト

ポートフォリオ用に作成した、架空の焼肉店サイトです。  
高級感のある焼肉店を想定し、和モダンな世界観とスマホ対応を意識して制作しています。

## 構成
- `index.html`
- `style.css`
- `script.js`

## 実装内容
- 1ページ完結型のランディングページ構成
- レスポンシブ対応
- ハンバーガーメニュー
- お知らせ一覧
- 店舗情報セクション
- ダミー予約フォーム
- フォーム送信時の簡易確認ダイアログ

## 想定ターゲット
- デート利用
- 家族での食事
- 会食利用
- 20代後半〜40代

## コンセプト
「一枚の旨さに、火が入る。」

## 使い方
`index.html` をブラウザで開くと表示できます。
